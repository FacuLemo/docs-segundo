## ¿Qué veremos en esta materia?

En esta materia nos enfocaremos en aprender a usar y crear APIs REST en el framework de python llamado FastAPI.

- Introducción a FastAPI; definición de conceptos clave
- Instalación de FastAPI y manejo de entornos virtuales
- Protocolo HTTP; Request y Responses en API REST
- Documentación Automática, Path Operations GET y POST, Parámetro de ruta y tipo query
- Path Operation Put/Patch y Delete, Body en lugar de query param
- Models de Pydantic; integrados a los path operations anteriores
- Clientes Rest como Postman, Bruno, etc.
- Conexión con Base de Datos
- SQLModel
- Autenticación segura de usuarios
- CORS
